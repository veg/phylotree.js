Welcome to Phylotree.js’s documentation!¶

Phylotree.js is a Javascript library for working with phylogenetic trees. Uses include web application development and interactive data visualization for phylogenetics.

It is written as a D3 layout, and aspires to be a general purpose tool for visualizing and selecting branches in phylogenetic trees. Example uses include a standalone web application for viewing and labeling phylogenetic trees, the Datamonkey webserver for comparative sequence analysis, an ancestral sequence structural viewer, and an interactive tanglegram.

The code is available on Github.
